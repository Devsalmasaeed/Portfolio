# Portfolio Website - Salma Abdullahi

## Description
This is a personal portfolio website for **Salma Abdullahi**, a Fullstack Developer specializing in modern web technologies. The portfolio showcases projects, skills, services, and contact information.

## Technologies Used
- **HTML5** - Structuring the website
- **CSS3** - Styling and responsive design
- **JavaScript (ES6+)** - Interactivity and animations
- **Boxicons** - Icons used in the UI
- **Typed.js** - Animated typing effect
- **ScrollReveal.js** - Scroll-based animations
- **Formspree** - Contact form handling

## Features
- **Fully responsive** across all devices
- **Interactive navigation bar** with smooth scrolling
- **Dynamic text animation** using Typed.js
- **Project showcase section** with live previews
- **Contact form** integrated with Formspree
- **SEO optimized** for better visibility

## Live Demo
Check out the live demo: [Portfolio Website](https://portfolio-viws.onrender.com)

## Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/Devsalmasaeed/Portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd portfolio
   ```
3. Open the `index.html` file in a browser or use a local server to run it.
